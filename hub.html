這個系列要解決什麼？

市面上自動化工具大多走「雲端服務 + 月費」路線：Zapier、Make、IFTTT 都是。但雲端版有三個痛點：每月要錢、資料在別人手上、跑不快。

這個系列教你把所有自動化工具蓋在自己電腦裡（地端部署），一次設好、永久免費、資料自己留。從最基礎的 Docker 安裝開始，一步一步擴充到接 Google、Notion、LINE 等服務。

這個系列適合你嗎？

適合這幾種人

第一種，想做自動化但不想付月費的人。你發現 Zapier 一個月幾百塊很痛，想找免費替代方案。n8n 自架版本，只要你電腦開著就免費跑到死。

第二種，資料隱私敏感的人。你的工作流程涉及客戶名單、敏感對話、私人筆記，不想交給雲端服務商存。地端部署，資料只在你硬碟上。

第三種，想學技術但被術語勸退過的人。我們用便當盒、廚房、房子當比喻，所有 UI 按鈕都用中文，每個步驟都先講「為什麼要做這件事」再講「怎麼做」。

不適合這幾種人

第一，需要團隊協作的中大型團隊。地端部署是個人或小團隊方案，多人協作建議直接用 n8n Cloud 或 self-host 在公司伺服器。

第二，不想動手摸技術的人。雖然我們把流程拆得很細，但仍然要打開 Docker、命令提示字元、GCP Console。如果這些字眼讓你頭痛，建議從 Zapier 或 Make 起步。
